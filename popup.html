Chzzk Syncer

치지직 싱크 컨트롤러 v0.2.4
현재 지연시간: -
← / → : 5초 이동
Ctrl + ← / → : 1초 이동
Shift + ← / → : 0.1초 이동
빨리감기 버튼: 최신 지점으로 이동

[사용법]
1. 방송화면 하단 왼쪽의 빨리감기나 팝업창의 초기화 버튼을 누른 후 빠른 쪽 방송을 클릭해주세요.
2. 방향키 혹은 팝업창의 숫자 변경 후 적용 버튼을 눌러 싱크를 맞춰주세요.
빨리감기 후 싱크를 맞춰도 어긋나는 상황이 반복될 때 체크 후 싱크를 다시 맞춰주세요. 사용자의 사양에 따라 버퍼링이 발생할 수 있습니다.
- 지연시간이 커지면 브라우저에서 자동으로 지연시간을 줄이는 보정을 실행하므로 싱크가 어긋날 수 있습니다.
- 현재 지연시간에 *가 붙어 있다면 현재 자동 보정이 이루어지고 있는 상태입니다. 빨리감기 후 다시 싱크를 조절해주세요.
